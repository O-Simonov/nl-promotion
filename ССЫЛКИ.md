## 4. UTM-метки — чтобы знать, ОТКУДА пришли люди

UTM — это «хвостик» к ссылке, который показывает в статистике, с какой площадки пришёл человек.
Полезно, когда у тебя будет **свой сайт/линк-хаб со статистикой** (например, через Google Analytics или встроенную статистику хостинга).

**Шаблон:** добавь к ссылке хвостик `?utm_source=ПЛОЩАДКА&utm_medium=social`

| Площадка | Готовая ссылка на линк-хаб с меткой |
|----------|-------------------------------------|
| Telegram | `https://oleg-nl.netlify.app/?utm_source=telegram&utm_medium=social` |
| ВКонтакте | `https://oleg-nl.netlify.app/?utm_source=vk&utm_medium=social` |
| YouTube | `https://oleg-nl.netlify.app/?utm_source=youtube&utm_medium=social` |
| TikTok | `https://oleg-nl.netlify.app/?utm_source=tiktok&utm_medium=social` |
| Одноклассники | `https://oleg-nl.netlify.app/?utm_source=ok&utm_medium=social` |
| Facebook | `https://oleg-nl.netlify.app/?utm_source=facebook&utm_medium=social` |
| MAX | `https://oleg-nl.netlify.app/?utm_source=max&utm_medium=social` |

> 📌 Важно: UTM работает на ТВОЁМ линк-хабе. Прямые реферальные ссылки nlstar.com лучше не «обвешивать» метками — они могут потеряться при переходе. Поэтому схема такая:
> **соцсеть → твой линк-хаб (с UTM) → реферальная ссылка.**
