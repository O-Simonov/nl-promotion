# 🤖 Telegram-автопостинг NL

Бот сам публикует по одному посту в день в твой Telegram-канал. Всё бесплатно, работает на твоём компьютере.

## Что внутри
| Файл/папка | Назначение |
|---|---|
| `config.json` | Токен бота и адрес канала (заполняется один раз) |
| `queue/` | Очередь постов (файлы `.txt`). Публикуются по порядку имён: 01, 02, 03… |
| `sent/` | Сюда переезжают уже опубликованные посты |
| `logs/post.log` | Журнал: что и когда опубликовано |
| `Test-Bot.ps1` | Проверка связи (запусти первым) |
| `Post-Next.ps1` | Публикует следующий пост из очереди |
| `Setup-Schedule.ps1` | Включает ежедневный автопостинг |

---

## 🚀 Настройка (один раз, ~5 минут)

### Шаг 1. Создай бота
1. В Telegram найди **@BotFather** и напиши ему `/newbot`.
2. Придумай имя и юзернейм бота (юзернейм должен заканчиваться на `bot`).
3. BotFather пришлёт **токен** — длинную строку вида `1234567890:AAH...`. Это ключ бота (не твой пароль).

### Шаг 2. Создай/выбери канал и добавь бота админом
1. Нужен **КАНАЛ** (не личный профиль). Если канала нет — создай (Telegram → ✏️ → «Создать канал»).
2. Открой канал → «Управление» → «Администраторы» → «Добавить» → найди своего бота → дай право **«Публикация сообщений»**.

### Шаг 3. Впиши данные в config.json
Открой `config.json` и вставь:
- `botToken` — токен из шага 1
- `channelId` — юзернейм канала, например `@Simka1969` (или числовой ID)

### Шаг 4. Проверь связь
В папке `tg-bot` запусти:
```
pwsh -File Test-Bot.ps1
```
Если в канал пришло «✅ Проверка связи» — всё работает.

### Шаг 5. Включи ежедневный автопостинг
```
pwsh -File Setup-Schedule.ps1 -Time 10:00
```
Готово! Каждый день в 10:00 бот публикует следующий пост из очереди.

---

## 📝 Как добавлять новые посты
Просто положи в папку `queue/` новый текстовый файл, например `08-den8.txt`.
- Файлы публикуются **по порядку имён**, поэтому нумеруй: 08, 09, 10…
- Хочешь пост **с картинкой** — положи рядом картинку с тем же именем: `08-den8.txt` + `08-den8.jpg`.

## ▶️ Опубликовать прямо сейчас, не дожидаясь расписания
```
pwsh -File Post-Next.ps1
```

## 🔒 Важно
- `config.json` с токеном — **не выкладывай в интернет** (он только на твоём компьютере, в папке site/ его нет).
- Если токен «утёк» — в @BotFather можно сделать `/revoke` и получить новый.
