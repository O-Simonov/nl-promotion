# VK-бот для автопостинга в сообщество ВКонтакте.

Работает в паре с Telegram-ботом: оба читают из общей очереди `../tg-bot/queue/`.
TG-бот берёт первый файл → публикует в 10:00 → переносит в `../tg-bot/sent/`.
VK-бот берёт первый файл → публикует в 14:00 → переносит в `sent/`.

## Файлы

- **`Post-Next-VK.ps1`** — публикация одного поста (читает очередь, шлёт в ВК, переносит в sent).
- **`Test-Bot-VK.ps1`** — проверка, что токен рабочий и сообщество доступно.
- **`Setup-Schedule-VK.ps1`** — создаёт задачу в Windows Task Scheduler (ежедневно в 14:00).
- **`config.json`** — токен и ID сообщества (НЕ коммитится).

## Настройка (один раз)

1. Создай сообщество ВК.
2. Зайди в **Управление → Настройки → Работа с API**.
3. Создай ключ с правами: `wall`, `photos`.
4. Скопируй токен в `config.json` (`botToken`).
5. Укажи ID сообщества в `config.json` (`groupId`, число без `club`).
6. Запусти `pwsh -File Test-Bot-VK.ps1` — должен увидеть имя сообщества.
7. Запусти `pwsh -File Setup-Schedule-VK.ps1` — задача в расписании.

## Важно

- **Скопируй** `config.example.json` в `config.json` и впиши свои значения.
- Никогда не пиши токен в чат/коммиты — он в `.gitignore`.
- Если токен утечёт — удали его в управлении сообщества и создай новый.
- Расписание работает **только когда ПК включён** (опция StartWhenAvailable срабатывает при следующем включении).
