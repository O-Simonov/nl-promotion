## Шаг 1. Посмотреть линк-хаб прямо сейчас

Просто **дважды кликни по файлу `index.html`** — он откроется в браузере. Это и есть твоя страница. Можно крутить, нажимать кнопки — всё работает локально.

---

## Шаг 2. Опубликовать в интернете (бесплатно)

Чтобы страница получила свою ссылку (которую ты дашь людям), её нужно «выложить». Три простых варианта — выбери любой:

### Вариант А — Netlify Drop (самый простой, 2 минуты) ⭐
1. Зайди на **https://app.netlify.com/drop**
2. Перетащи файл `index.html` прямо в окно браузера.
3. Готово! Тебе дадут ссылку вида `https://nazvanie123.netlify.app`.
4. Можно бесплатно поменять название на красивое (в настройках сайта → *Change site name*).

### Вариант Б — GitHub Pages (надёжно, навсегда)
1. Создай бесплатный аккаунт на **https://github.com**
2. Создай репозиторий с именем `nl` (или любым).
3. Загрузи `index.html`.
4. Settings → Pages → выбери ветку `main` → Save.
5. Через минуту страница будет по адресу `https://твой-логин.github.io/nl/`.

### Вариант В — Taplink / mssg.me (визуальный конструктор, без файлов)
Если не хочешь возиться с файлами — есть русскоязычные конструкторы «ссылки в шапку профиля» (Taplink, mssg.me). Там можно собрать похожую страницу мышкой. Минус — часть функций платная. Наш `index.html` бесплатен и полностью твой.

---

## Шаг 3. Поставить ссылку везде

Когда получишь адрес линк-хаба — вставь его в **шапку / описание** каждого профиля:

- **Telegram** — в описание канала/профиля
- **ВКонтакте** — в блок «Ссылки» или статус
- **YouTube** — «О канале» → ссылки
- **TikTok** — поле «Веб-сайт» в профиле
- **Одноклассники** — в профиль
- **Facebook** — в раздел «Контакты»
- **MAX** — в описание профиля

📌 **Идея:** во всех постах пиши «все ссылки в шапке профиля» или «ссылка в описании» — это и удобно, и не выглядит как спам.

---

## Шаг 4. Поменять фото и текст (по желанию)

Открой `index.html` в Блокноте (правый клик → «Изменить») и найди:

- **Имя** — строка `<h1>Олег Симонов</h1>`
- **Подпись под именем** — блок `class="tagline"`
- **Фото вместо букв «ОС»** — инструкция прямо в коде, ищи комментарий `Чтобы поставить своё фото`

После изменений сохрани файл и заново перетащи на Netlify (Вариант А) — обновится.
