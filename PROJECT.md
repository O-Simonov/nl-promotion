# Описание проекта NL AutoPost

## Название
**NL AutoPost** — автопубликация постов о продуктах NL International в 4 соцсети по расписанию.

## Проблема
Дистрибьютор NL International ведёт несколько соцсетей параллельно (Telegram, ВКонтакте, Facebook, Instagram). Ручной постинг ежедневно отнимает 30–60 минут, легко пропустить день, а регулярность — ключевой фактор охвата и продаж. Платные сервисы автопостинга (SMMplanner, Postmypost и т.п.) либо не поддерживают ВК-сообщества + IG одновременно, либо стоят дорого для партнёра с небольшим товарооборотом.

## Пользователь
- **Основной:** дистрибьютор NL (Олег Симонов) — использует систему для ежедневной публикации в свои 4 канала.
- **Дополнительный:** другие партнёры NL, которые поднимают свою копию через `Setup-Partner.ps1` и получают ту же автоматизацию со своими реф-ссылками и контентом.

## Основная функция
- **Вход:** папка с постами в формате `.txt` + картинка `.png` того же имени. Контент лежит в `queue/` (на публикацию) и `backlog/` (запас). Тексты и обложки готовятся вручную или через встроенный генератор картинок.
- **Выход:** опубликованный пост в Telegram-канале, ВК-сообществе, Facebook-странице, Instagram-аккаунте. Все 4 площадки получают по одному посту в день со сдвигом по времени.
- **Инструменты:** PowerShell 7-скрипты, Windows Task Scheduler (планировщик задач), API соцсетей (Telegram Bot API, VK Open API, Meta Graph API), ImgBB для хостинга картинок, Netlify для статического сайта-хаба.
- **Дополнительно:** автопополнение очереди из бэклога, когда постов остаётся ≤ 8; утренний и вечерний сбор новостей NL из Telegram-канала `@nl25_news` для идей постов.

## Формат результата
- **Для дистрибьютора:** готовые опубликованные посты с реферальными ссылками на продукцию NL — без ежедневных ручных действий. Лог-файлы в `*/logs/post.log` показывают историю публикаций.
- **Для подписчиков:** обычный пост в ленте: текст + обложка + ссылка на продукт / каталог.
- **Сайт-хаб** на Netlify (`oleg-nl.netlify.app`) — каталог продуктов, презентация компании, инструкция «как заказать», страница «бизнес с NL».
- **Для партнёра NL:** клон проекта, в котором за 5 минут (`Setup-Partner.ps1`) подставляются его имя, соцсети и реф-ссылки.

## Минимальный функционал (MVP) — что уже работает
- ✅ 4 бота (Telegram, ВКонтакте, Facebook, Instagram) с постингом по расписанию
- ✅ Windows Task Scheduler: 09:00 refill → 10:00 TG → 11:00 FB → 12:00 IG → 14:00 VK
- ✅ Автопополнение очереди из `backlog/` (Check-Queue-Refill)
- ✅ Генератор картинок `Make-PostImage.ps1` (12 категорий: product, business, review, offer, news, slimming, hair, skincare, body, mens, kids, teeth)

- ✅ News-Watch: сбор новостей из TG `@nl25_news` (08:30 + 20:00) → `digest.md`
- ✅ Режим партнёра: `Setup-Partner.ps1` подставляет свои реф-ссылки, соцсети, имя

## Технологии
- **Язык:** PowerShell 7+ (pwsh). Выбран ради штатной автоматизации Windows Task Scheduler и нулевых внешних зависимостей.
- **Планировщик:** Windows Task Scheduler (встроен в Windows 10/11).
- **API:**
  - Telegram Bot API (`api.telegram.org`)
  - VK Open API (`api.vk.com`)
  - Meta Graph API (`graph.facebook.com`) — единый токен для FB и IG
- **Хостинг картинок:** ImgBB (бесплатный, безлимит на размер; ссылка подставляется в API соцсетей).
- **Сайт:** статический HTML + Netlify (continuous deployment из GitHub).
- **Форматы:** `.txt` для текста поста, `.png` / `.jpg` для обложки, `.json` для конфигов, `.md` для базы знаний и контент-плана.
- **Хранилище данных:** файловая система (`queue/`, `backlog/`, `sent/`, `logs/`). Никаких БД.
